# Projeto_TC

Este app carrega modelos pré-treinados para realizar previsões de tendência do fechamento de ações ou índices da B3.

O Notebook treina_modelo_1_1.ipynb pode ser utilizado para treinar o modelo para novas datas, alterando suas configurações. O arquivo Acao.py carrega a classe que faz o download do dados financeiros a partir da biblioteca yfinance.
